# Inventory & Order Management System

A simplified full-stack application with:

- FastAPI Backend (Python)
- React Frontend
- PostgreSQL
- Docker + Docker Compose

## Features

- Unique Product SKUs
- Unique Customer emails
- Inventory validation & automatic stock reduction
- Responsive UI
